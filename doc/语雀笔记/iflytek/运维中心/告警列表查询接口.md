# 告警列表查询接口

## 1.接口认证

接口地址：`/devops/sys-user/login`

请求方式：GET

请求参数：

|参数名|参数类型|是否必传|说明|
| --------| --------| --------| -----------------------|
|userName|String|是|用户名|
|password|String|是|密码，需要经过 md5 加密|
|code|String|是|默认值 iflytek|

响应参数：

```json
{
  "code": 0,
  "msg": "操作成功",
  "data": {
    "avatar": "",
    "createBy": "iflytek",
    "createTime": "2021-02-23 14:36:54",
    "delFlag": 0,
    "email": "",
    "loginDate": 1614062219000,
    "loginIp": "",
    "loginName": "iflytek",
    "password": "",
    "phonenumber": "",
    "salt": "iflytek",
    "sex": "0",
    "token": "0500a765-067e-4319-9310-4ef97109d7b7",
    "updateBy": "",
    "userId": 1,
    "userName": "iflytek",
    "userStatus": 0,
    "userType": "00"
  }
}
```

## 2.接口调用

拿到 1 中接口认证中的 token 参数，并携带在请求头中，例如：

​`Token: 0500a765-067e-4319-9310-4ef97109d7b7`​

然后参考**​`运维中心v2.5-监控平台接口文档--2.7.13 告警列表查询接口`​**​ **，** 进行接口调用即可
