# 二叉树的序列号与反序列化

## 1.题目描述

> 序列化是将一个数据结构或者对象转换为连续的比特位的操作，进而可以将转换后的数据存储在一个文件或者内存中，同时也可以通过网络传输到另一个计算机环境，采取相反方式重构得到原数据。
>
> 请设计一个算法来实现二叉树的序列化与反序列化。这里不限定你的序列 / 反序列化算法执行逻辑，你只需要保证一个二叉树可以被序列化为一个字符串并且将这个字符串反序列化为原始的树结构。

> 示例:

> 你可以将以下二叉树：

>    1

>   / \
>
>  2   3
>
>      / \
>
>     4   5

> 序列化为 "[1,2,3,null,null,4,5]"

> 提示: 这与 LeetCode 目前使用的方式一致，详情请参阅 LeetCode 序列化二叉树的格式。你并非必须采取这种方式，你也可以采用其他的方法解决这个问题。

## 2.解决思路

**序列化：** 首先按照层序遍历的方式遍历二叉树，从根结点开始存储所有非空结点及其左右孩子到集合中，例如上述二叉树遍历后得到的结果就是`[1, 2, 3, null, null, 4, 5, null, null, null, null]`，以此作为序列号的结果。

**反序列化：** 对于上述结果可以看出处了第一个元素为根结点外，剩余元素都是作为左右子结点成对出现，因此可以构造队列按层次反序列化的到员树。

> - 取出第一个元素，构造根结点->根结点入队
> - 遍历队列
> - 取出队头节点，取出集合中前两个元素构造结点，并作为队头元素的左右孩子（需要判断是否为null），同时左右孩子入队
> - 继续遍历队列，知道队为空，返回头结点

### 2.1Java代码

```java
/**
 * Definition for a binary tree node.
 * public class TreeNode {
 *     int val;
 *     TreeNode left;
 *     TreeNode right;
 *     TreeNode(int x) { val = x; }
 * }
 */
public class Codec {
    // Encodes a tree to a single string.
    public String serialize(TreeNode root) {
        if (root == null) return null;
        Queue<TreeNode> queue = new LinkedList<>();
        List<Integer> res = new ArrayList<>();
        queue.add(root);
        while (!queue.isEmpty()) {
            int n = queue.size();
            for (int i = 0; i < n; i++) {
                TreeNode p = queue.poll();
                if (p != null) {
                    queue.add(p.left);
                    queue.add(p.right);
                }
                res.add(p == null ? null : p.val);
            }
        }
        String s = res.toString();
        return s.substring(1, s.length() - 1);
    }

    // Decodes your encoded data to tree.
    public TreeNode deserialize(String data) {
        if (data == null) return null;
        String[] split = data.split(",\\s+");
        Queue<Integer> queue = new LinkedList<>();
        for (String s : split) {
            if ("null".equals(s)) {
                queue.add(null);
            } else {
                queue.add(Integer.parseInt(s));
            }
        }
        Queue<TreeNode> temp = new LinkedList<>();
        Integer rootVal = queue.poll();
        TreeNode root = rootVal == null ? null : new TreeNode(rootVal);
        temp.add(root);
        while (!temp.isEmpty()) {
            TreeNode p = temp.poll();
            Integer leftVal = queue.poll();
            Integer rightVal = queue.poll();
            if (leftVal != null) {
                TreeNode left = new TreeNode(leftVal);
                p.left = left;
                temp.add(left);
            }
            if (rightVal != null) {
                TreeNode right = new TreeNode(rightVal);
                p.right = right;
                temp.add(right);
            }
        }
        return root;
    }
}
```
