# 移0问题_

> 给定一个数组nums，写一个函数，将数组中所有的0挪到数组的末尾，⽽维持其他所有非0元素的相对位置。
>
> 举例: nums = [0, 1, 0, 3, 12]，函数运⾏后结果为[1, 3, 12, 0, 0]

### 1.解决思路1

> 维护一个临时变量k=0，遍历数组将非0元素移到k的位置并将k后移一位，遍历完成后将k到nums.length的元素置为0即可

#### 1.1Java代码

```java
private static void solution() {
    int[] a = {0, 1, 0, 3, 12};
    // 临时变量k
    int k = 0;
    // 遍历数组，将非0元素移到k的位置，并将k后移一位
    for (int i = 0; i < a.length; i++) {
        if (a[i] != 0) {
            a[k++] = a[i];
        }
    }
    // 将k到a.length的元素置为0
    for (int i = k; i < a.length; i++) {
        a[i] = 0;
    }
    Arrays.stream(a).forEach(System.out::println);
}
```

### 2.解决思路2

> 维护一个临时变量k=0，遍历数组并交换nums[i]和nums[k]，同时k后移一位，交换条件如下：
>
> 1. nums[i] != 0
> 2. i != k，否则交换没有意义，如果相等则k+1

#### 2.1Java代码

```java
private static void solution() {
    int[] a = {0, 1, 0, 3, 9};
    // 临时变量k
    int k = 0;
    // 遍历数组
    for (int i = 0; i < a.length; i++) {
        // 当前元素不为0是交换的初始条件
        if (a[i] != 0) {
            // k 和 i相等交换没有意义
            if (k != i) {
                // 在该条件下由于k 始终 小于 i，所以k一定指向0元素
                a[k] = a[i];
                a[i] = 0;
                k++;
            }else {
                k++;
            }
        }
    }
    Arrays.stream(a).forEach(System.out::println);
}
```
