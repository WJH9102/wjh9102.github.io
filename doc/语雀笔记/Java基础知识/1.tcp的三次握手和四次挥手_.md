# 1.tcp的三次握手和四次挥手_

## 1.三次握手（建立连接）

1. 客户端向服务端发送请求询问是否能接收到
2. 服务端回应可以接收到，并询问客户端能否接收到
3. 客户端回应可以接收到 ------>通信

![](assets/network-asset-20200515144412-20250108101729-jx1m1mx.png)

## 2.四次挥手（关闭连接）

1. 客户端发送关闭连接请求
2. 服务端回应收到请求
3. 服务端处理数据并向客户端发送可以关闭请求
4. 客户端关闭连接
