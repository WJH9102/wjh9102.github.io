# 泛型

1. 实现泛型接口时实现类必须添加与泛型接口一致泛型，或者都不添加，则默认为Object类型

```java
public interface FatherInterface<E> {
    void show(E e);
}

class SonClass<E> implements FatherInterface<E> {
    @Override
    public void show(E o) {
        System.out.println(o);
    }
}
```

2. 继承泛型类时需要知道父类的泛型实参，或者子类添加与父类一致的泛型，或者都不添加，则父类泛型类型m默认为原始类型

```java
public class FatherClass<E> {
}
class SonCls<E> extends FatherClass<E>{
}
```
